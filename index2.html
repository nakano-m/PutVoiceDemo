PutVoice 〜声を残していこう〜

Created by Masahiro Nakano / Kazuki Morita

やりたいこと

  • Webで声を保存・共有したい

背景

  • 時々タイピングが面倒
  • 女の声が聞きたい
  • 声で考えを共有したい
  • ※声優を絡めたい^^

選定した技術(client側)

Web Audio API

  • Flashとかだるい、アクセシビリティがががが(略)

選定した技術(サーバー側)

Play framework(java)

  • とっつきやすい
  • 森田さんJavaの経験あるので

想定する機能概要

  • ログイン/ユーザー作成機能
  • 録音/再生機能
  • 音声保存機能
  • 指定した場(ユーザー含め)に録音音声を置く

進捗状況

  • 音声録音/再生機能はなんとかできた
  • ファイルをアップロードできた

操作はとても簡単

  • 録音ボタンを押してマイクに言葉を言い続けるだけ
  • 終わったら停止ボタンクリック

ブログでタイプするのも大変な人もできたりします

  • ブログ更新したいけど、時間ないし打つ時間も減らしたい時など
  • たまにはそういう更新もいいんじゃないでしょうか

デモ

苦労した点

  • ブラウザのサポートがまだ。。。
  • マイクが必要(今時のPCにはついているはず…！)
  • Web AudioのAPIがクソ(グローバル変数氏〜)
